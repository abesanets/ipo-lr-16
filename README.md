# Интернет-магазин электроники

## Описание

Учебный проект на Django, созданный в рамках лабораторных работ.
Приложение представляет собой полнофункциональный интернет-магазин электроники
с каталогом товаров, корзиной, системой аутентификации, оформлением заказов
и автоматической отправкой чеков на email.

## Функционал

### Общедоступные страницы
- **Главная страница** — приветствие, категории, последние товары
- **Каталог товаров** — список товаров с фильтрацией, поиском и сортировкой
- **Карточка товара** — подробная информация о товаре
- **О магазине** — информация о магазине и статистика
- **Об авторе** — информация о разработчике

### Для авторизованных пользователей
- **Корзина** — управление товарами (добавление, изменение количества, удаление)
- **Оформление заказа** — форма с данными доставки
- **История заказов** — просмотр оформленных заказов
- **Автоматическая генерация чека** — Excel-файл с составом заказа
- **Отправка чека на email** — после оформления заказа

### Административная панель
- Управление товарами, категориями, производителями
- Управление пользователями и их корзинами
- Просмотр и управление заказами

## Модели базы данных

### Категория товара (Category)
- Название, описание

### Производитель (Manufacturer)
- Название, страна, описание

### Товар (Product)
- Название, описание, фото, цена, количество на складе
- Связь с категорией и производителем
- Валидация: цена и количество ≥ 0

### Корзина (Cart)
- Связь с пользователем (один к одному)
- Дата создания
- Метод подсчёта общей стоимости

### Элемент корзины (CartItem)
- Связь с корзиной и товаром
- Количество, стоимость элемента
- Валидация: количество не превышает остаток на складе

### Заказ (Order)
- Связь с пользователем
- Адрес доставки, телефон, email
- Дата создания, общая стоимость

### Элемент заказа (OrderItem)
- Связь с заказом
- Название товара, цена, количество
- Стоимость элемента


**macOS/Linux:**
```bash

```

### 4. Установка зависимостей

```bash
pip install django Pillow openpyxl
```

### 5. Настройка email (важно!)

Откройте файл `shop_project/settings.py` и найдите раздел **Email настройки**.

Замените следующие строки **на свои данные**:

```python
# Email настройки (Gmail)
EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
EMAIL_HOST = 'smtp.gmail.com'
EMAIL_PORT = 587
EMAIL_USE_TLS = True
EMAIL_HOST_USER = 'ВСТАВЬТЕ_ВАШУ_ПОЧТУ@gmail.com'        # ← ваш Gmail
EMAIL_HOST_PASSWORD = 'ВСТАВЬТЕ_ВАШ_ПАРОЛЬ_ПРИЛОЖЕНИЯ'   # ← пароль приложения Gmail
DEFAULT_FROM_EMAIL = 'ВСТАВЬТЕ_ВАШУ_ПОЧТУ@gmail.com'
```

#### Как получить пароль приложения Gmail:

1. Перейдите в [Google Аккаунт](https://myaccount.google.com/)
2. Безопасность → Двухэтапная аутентификация (включите, если не включена)
3. Пароли приложений → Выберите "Почта" и "Другое устройство"
4. Скопируйте сгенерированный пароль (16 символов)
5. Вставьте в `EMAIL_HOST_PASSWORD`

> **Альтернатива для тестирования:** Если не хотите настраивать Gmail, замените `EMAIL_BACKEND` на:
> ```python
> EMAIL_BACKEND = 'django.core.mail.backends.console.EmailBackend'
> ```
> Письма будут выводиться в консоль терминала вместо отправки.

### Оформление заказа

1. Войдите в систему
2. Перейдите в **Каталог**
3. Добавьте товары в корзину
4. Перейдите в **Корзину** → **Оформить заказ**
5. Заполните форму:
   - Адрес доставки
   - Телефон
   - Email (на него придёт чек)

## Возможные проблемы и решения

### Письма не отправляются
Проверьте `settings.py`:
- Правильность email и пароля приложения Gmail
- Включена ли двухэтапная аутентификация в Google
- Для теста используйте `console.EmailBackend`

### Товары не добавляются в корзину
Убедитесь, что:
- Вы авторизованы
- Товар есть в наличии (stock > 0)
