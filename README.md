## Функционал

### Общедоступные страницы
- **Главная** — hero-секция, категории, последние товары
- **Каталог** — фильтрация по категории, производителю, цене; поиск, сортировка, пагинация (9 товаров/стр.)
- **Карточка товара** — фото, описание, цена, наличие, кнопка "В корзину"
- **О магазине** — статистика (товары, категории, производители)
- **Об авторе** — информация о разработчике

### Для авторизованных пользователей
- **Корзина** — добавление, изменение количества, удаление товаров
- **Оформление заказа** — форма с адресом, телефоном, email
- **Автоматическая генерация чека** — Excel-файл с составом заказа
- **Отправка чека на email** — после оформления заказа

### REST API
- Полный CRUD для всех моделей через Django REST Framework
- Публичное чтение товаров, категорий, производителей (без авторизации)
- Корзина и заказы — только для авторизованных пользователей
- Browsable API: `/api/`

### Клиентский интерфейс (JS)
- Динамическая загрузка товаров через Fetch API (`/api/products/`)
- Добавление в корзину через POST-запрос с CSRF-токеном
- Bootstrap Toast-уведомления после добавления товара
- Спиннер загрузки во время запроса к API
- Обработка ошибок (403, 500, сеть)

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <url-репозитория>
```

### 2. Создание и активация виртуального окружения

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install django djangorestframework Pillow openpyxl
```

### 4. Настройка email

В `shop_project/settings.py` замените на свои данные:

```python
EMAIL_HOST_USER = 'ваша_почта@gmail.com'
EMAIL_HOST_PASSWORD = 'пароль_приложения'  # 16-символьный пароль приложения Gmail
DEFAULT_FROM_EMAIL = 'ваша_почта@gmail.com'
```

Для тестирования без отправки писем:
```python
EMAIL_BACKEND = 'django.core.mail.backends.console.EmailBackend'
```

### 5. Применение миграций и запуск

```bash
python manage.py migrate
python manage.py runserver
```

Откройте: **http://127.0.0.1:8000/**
